# Geopolymer Clogging ML GUI

A unified machine-learning-based GUI to predict:
- Porosity (%)
- Permeability (mm/hr)
- Clogging rate (% per year)

## How to Run
1. pip install -r requirements.txt
2. python train_models.py
3. streamlit run app.py

## Notes
- Trained models (*.pkl) are generated locally
- Input data is not included in the repository
